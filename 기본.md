# C 프로그램의 기본 
- C 프로그램의 구성 요소
  - 주석
  - main 함수
  - 출력
- C 언어의 입력과 출력
  - 변수
  - printf 함수
  - scanf 함수
  
# 주석
- 주석을 다는 방법
  - /*과 */을 이용한 여러 줄 주석
  - //을 이용한 한 줄 주석
- 주석의 용도
  - 프로그램에 대한 설명
  - 프로그램 전체에 대한 대략적인 정보를 제공
  - 주석 처리(comment out)// 실행되지 않게 한다. 

# 문장
- C 프로그램을 이루는 기본 단위
- 각 문장은 세미콜론(;)으로 끝난다.
- ~문: 선언문, 입력문, 출력문, 조건문, 반복문 등
